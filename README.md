# Tetris 3.0

## Created using Python and Tkinter

This game was inspired when watching the 'move_rectangle' lecture from Stanford's Code In Place 2020.

## How to Play

* Use arrow keys to move tetromino
* Up arrow to rotate
* Space bar to hard drop
